### Приклади для запуску програми:
```
python prog.py 1 3 5 4.6 
``` 
```
python prog.py -f input.txt
``` 
```
python prog.py -f input.txt -o result.txt 
``` 
```
python prog.py -o result.txt -- 1 5 -2
``` 
аби від'ємні числа (-2 в нашому випадку) не сприймалися як параметр ми пишемо -- після параметрів з файлами, а далі - цифри.
